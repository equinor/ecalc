## Description {/* #description */}
Specifies the path (relative or absolute) to the input file.

The path should be a reference to a [CSV](https://en.wikipedia.org/wiki/Comma-separated_values) file.

## Format {/* #format */}
~~~~~~~~yaml
FILE: </path/to/file.csv>
~~~~~~~~
