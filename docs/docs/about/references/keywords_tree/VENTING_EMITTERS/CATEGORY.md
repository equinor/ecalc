## Description
Venting emitter [CATEGORY](/about/references/keywords/CATEGORY.md), 
typically `COLD-VENTING-FUGITIVE`, `LOADING` or `STORAGE`. 

## Format
~~~~~~~~yaml
VENTING_EMITTERS:
  - NAME: <name>
    CATEGORY: <category>
~~~~~~~~

## Example

~~~~~~~~yaml
VENTING_EMITTERS:
  - NAME: SomeVentingEMitter
    CATEGORY: COLD-VENTING-FUGITIVE
~~~~~~~~
