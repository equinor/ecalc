## Description
Used to define the fluid density for
pump type `energy usage models<ENERGY_USAGE_MODEL>`.

## Format
~~~~~~~~yaml
FLUID_DENSITY: <fluid density value or expression>
~~~~~~~~

## Example
~~~~~~~~yaml
FLUID_DENSITY: 1030 # [kg/m3]
~~~~~~~~
