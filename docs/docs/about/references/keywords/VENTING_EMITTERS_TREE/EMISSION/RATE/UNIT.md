## Description
Optional. Emission rate unit. Available units:
- `kg/d` (kilo per day, default)
- `t/d` (tons per day)

## Format
~~~~~~~~yaml
...
RATE:
  UNIT: <unit>
~~~~~~~~

## Example

~~~~~~~~yaml
...
RATE:
  UNIT: kg/d
~~~~~~~~
