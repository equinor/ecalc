## Description
Optional. Emission rate type. Available rate types:
- `STREAM_DAY` (default)
- `CALENDAR_DAY`

## Format
~~~~~~~~yaml
...
RATE:
  TYPE: <rate type>
~~~~~~~~

## Example

~~~~~~~~yaml
...
RATE:
  TYPE: STREAM_DAY
~~~~~~~~
